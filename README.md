# Where-Should-You-Work
UCSD GPGN 490 Final Project

## Overview

This project develops an interactive Shiny dashboard that helps users compare U.S. states based on wages, housing costs, unemployment rates, and cost of living.

The goal is to support location decisions for recent graduates, early-career workers, and job seekers by showing that a higher salary does not always translate into greater purchasing power.

## Dashboard

Live App:

https://zhangcx.shinyapps.io/WhereShouldYouWork/

## Data Sources

* Bureau of Labor Statistics (OEWS): Average wages by state
* Bureau of Labor Statistics (LAUS): State unemployment rates
* U.S. Census Bureau American Community Survey (ACS): Median gross rent
* Bureau of Economic Analysis (BEA): Regional Price Parities (RPP)

## Key Variables

* Average Salary
* Median Gross Rent
* Unemployment Rate
* Regional Price Parity (Price Index)
* Real Salary (cost-of-living adjusted salary)
* Rent Share (% of salary spent on rent)
